🚀 trash-api

trash-api is a social media scraper and downloader library for Node.js.
It allows you to retrieve information, search for content, and download media from multiple platforms quickly and easily.

Ideal for automation projects, bots, and scraping tools.

---

✨ Features

· 📥 Download content from multiple platforms
· 🔎 Search scrapers
· 🧰 Automation utilities
· ⚡ Fast and lightweight
· 📦 Easy to integrate into Node.js projects
· 🤖 Perfect for WhatsApp, Discord, or Telegram bots

---

📦 Installation

Install the package from npm:

```bash
npm install trash-api
```

---

🔧 Requirements

· Node.js 18 or higher

---

🧪 Usage example

```js
const trashcore = require("trash-api")

async function run(){

 let result = await trashcore.tiktok("https://vt.tiktok.com/ZSuh9Q5Ug/")

 console.log(result)

}

run()
```
